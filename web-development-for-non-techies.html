Web development for non techies

Learn about web development without learning to program!

This course is aimed at people working with developers: project managers, recruiters and non-technical cofounders.
Instead of showing how technologies are used the course describes what the different technologies do and how they relate to each other.

I started writing this course in November 2014. You can sign up for updates below.
